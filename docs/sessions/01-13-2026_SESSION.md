## Key Decisions & Why

- **CSV Format Detection**: `auto_detect_format()` allows CSV-first implementations to support both legacy and real university datasets
- **Config as YAML**: Industry-standard format with JSON fallback for environments without PyYAML
- **Instrumentation Split**: Separate `Statistics` and `Tracing` modules allow optional debug logging without overhead
- **Flattened Data Structure**: Flat CSV files simplify CI/CD, testing, and data pipeline management
